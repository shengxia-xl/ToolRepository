关于stl中容器的线程安全

多个读取者是安全的
对不同容器的多个写入者是安全的。

部分方法是安全的 
其大意也是，为了效率，没有给所有操作加锁。

----------------------------------------------------------------------------------------------------
STL中的容器分为

vector 
动态数组(大小不够时，会重新计算) 存储地址是连续的

list
线性链表 

deque
与vector类似，添加了从头插入数据的方法，存储地址是一部分一部分连续的

set
map
都是由红黑树来实现的 

hashset hashmap 是由hash实现的。
